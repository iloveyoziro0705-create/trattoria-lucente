========================================
 対応フォーマット
========================================
  .jpg / .jpeg → 推奨（軽量・Webに最適）
  .png         → 透過が必要な場合
  .webp        → 最も軽量（モダンブラウザ対応）
  .mp4         → 動画（hero.mp4 のみ）

  ※ 写真は500KB〜1MB以内に圧縮推奨
     圧縮ツール: https://squoosh.app/（無料・ブラウザで使用可）
  ※ 動画は10〜20MB以内を目安に
     変換・圧縮ツール: Handbrake（無料）
